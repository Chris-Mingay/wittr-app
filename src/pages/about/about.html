About
Why? On Friday the 12th of February 2016, Simon read an email by Ian, an L.T.L. F.T.E. from Christchurch NZ asking if any app developers out there would be able to make a mobile app to connect nearby listeners. Granted, it was requested for Colonial Commoners, but I didn't want to exclude those currently residing within the British Empire. Note: This is nothing to do with the BBC, they wanted me to let you know that! What exactly does it do? Basically, if you've got this app installed and you've enabled it in the settings screen, it will run off pin you on a map of Wittertainees. The positions are all "fuzzed up" a bit, so you'll never be giving your exact location away, but it should be a fun way to see if others Wittertainees are in the local area. There's also a copy of the Code of Conduct, should you need to reference it (but we all know it off the top of our heads now... right?). What WILL it do? One part of the request was for the app to let out a little "Hello to Jason Isaacs" when someone else with the app was nearby. While not implemented yet, this will be possible... maybe... I dunno, it seems like something I should disable if in the proximity of a cinema....
